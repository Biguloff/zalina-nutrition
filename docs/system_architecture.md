# Архитектура системы управления клиентами
## Залина Бигулова — Нутрициолог-натуропат

> Документ описывает целевую систему: хранение данных, структуру папок, базу данных, карточку клиента, автоматизацию. Цель — превратить ручной процесс (1200 сообщений/мес на клиента) в полуавтоматический бизнес.

- Глубина вложенности: максимум 3 уровня
- При завершении программы → перенос в `02_Клиенты_Архив/YYYY/`
